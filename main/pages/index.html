📌 Sobre o Projeto

O CareHub é um sistema completo que permite aos usuários agendarem consultas, registrarem emergências e acompanharem dicas de saúde. O projeto foi desenvolvido em grupo como parte do Projeto Integrador V do curso de Análise e Desenvolvimento de Sistemas do SENAC.

🚀 Tecnologias Utilizadas

  • Frontend: ReactJS
  • Backend: Node.js com Express
  • Banco de Dados: PostgreSQL (Tembo)

📂 Funcionalidades

  • Login e Cadastro de Usuário
  • Agendamento de Consultas
  • Registro de Fichas de Emergência
  • Dicas de Saúde Personalizadas
  • Área Administrativa com Gerenciamento de Dados
